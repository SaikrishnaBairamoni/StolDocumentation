The Zephyr OS is a small-\/footprint kernel designed for use on resource-\/constrained and embedded systems\+: from simple embedded environmental sensors and LED wearables to sophisticated embedded controllers, smart watches, and IoT wireless applications.

See the \href{https://zephyrproject.org}{\texttt{ zephyrproject.\+org}} site for more information about the project and becoming a member, and this \href{https://docs.zephyrproject.org/latest/introduction/index.html\#resources}{\texttt{ summary of Zephyr resources}} to help you find your way around the project.\hypertarget{md_doc__doxygen_mainpage_autotoc_md1}{}\doxysection{Supported Boards}\label{md_doc__doxygen_mainpage_autotoc_md1}
